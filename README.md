# Treasure Hunt App

A location-based mobile treasure hunt game for Sacramento, CA. Explore the city, discover hidden treasures, and compete with friends in this GPS-enabled adventure app built with Android Studio and Kotlin.

## Features

- **Real-time GPS Tracking** - Uses device location to track your position in real-time
- **Interactive Map** - View treasure locations and your proximity to them
- **Treasure Discovery** - Find hidden treasures at real locations around Sacramento
- **Progress Tracking** - Keep track of discovered treasures and overall completion
- **Distance Calculation** - See how far you are from each treasure location
- **Achievement System** - Unlock rewards as you discover more treasures

## How It Works

1. **Start the Hunt** - Launch the app and grant location permissions
2. **Explore Sacramento** - Travel to different locations around the city
3. **Find Treasures** - When you get close enough to a treasure location, you can claim it
4. **Track Progress** - View your collection and see which treasures you've found
5. **Complete the Hunt** - Discover all treasures to complete the challenge

## Prerequisites

- Android Studio (latest version recommended)
- Android device or emulator with GPS capabilities
- Minimum Android SDK: API level 21 (Android 5.0 Lollipop) or higher
- Google Maps API key

- **ACCESS_COARSE_LOCATION** - For approximate location
- **INTERNET** - For loading map data
